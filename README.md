# Gmod-Script-Suite
A set of random scripts packed together.

## Includes:
* Timer for rounds/map rotation/events
* Dynamic server name changing

## How to install
Extract files to server's addons folder. File strcuture should look like this.

```
addons/
└ racc-script-suite/
  └ lua/
    ├ autorun/
    ├ ─ server/
    |    └ init.lua
    ├ racc-script-suite/
      ├  config.lua
      └ scripts/
         ├ RaccTimer.lua
         └ DynamicName.lua
  ```
